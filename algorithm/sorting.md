## 1. 排序问题总论

定义排序的稳定性、原地性、自适应性等评估维度，给出基于比较的排序算法下界 Ω(n log n) 的决策树证明。

## 2. 冒泡排序

阐述相邻元素交换的冒泡过程，分析最好 O(n)、最坏 O(n²) 情况，讨论提前终止优化与鸡尾酒排序变种，附逐轮可视化图与 Python / C++ / Java 实现。

## 3. 选择排序

讲解每轮选取最小元素放到前端的策略，证明无论输入如何时间复杂度恒为 Θ(n²)，说明其不稳定版本的修复方法，附可视化与三语言实现。

## 4. 插入排序

描述将元素逐个插入已排序区间的过程，分析最好 O(n)（近乎有序）、最坏 O(n²) 的场景，讨论二分插入优化与 Shell 排序的间隔序列，附可视化与三语言实现。

## 5. 快速排序

详解 Lomuto 与 Hoare 两种分区方案，分析平均 O(n log n) 与最坏 O(n²) 的触发条件，讲解随机化、三路分区、内省排序等优化策略，附递归树可视化与三语言实现。

## 6. 归并排序

阐述分治归并的核心思想，证明时间复杂度恒为 Θ(n log n) 及空间 O(n)，讨论自顶向下与自底向上两种实现方式，附合并过程可视化与三语言实现。

## 7. 堆排序

回顾堆的 sift-down / sift-up 操作，讲解建堆 O(n) 的证明与堆排序流程，分析原地排序但非稳定的特性，附堆调整可视化与三语言实现。

## 8. 综合对比与选择

用表格横向对比六种排序的时间/空间/稳定性/适应性，给出不同场景（小规模、近乎有序、要求稳定、内存受限）下的算法选择建议。
