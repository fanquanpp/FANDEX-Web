## 1. 搜索问题总论

区分静态搜索与动态搜索、精确匹配与近似匹配，介绍搜索问题的形式化定义与评估指标（查找成功/失败的平均比较次数）。

## 2. 线性搜索

讲解顺序扫描策略，分析无序数组 O(n) 与带哨兵优化的线性搜索，讨论在链表等非随机访问结构上的适用性，附 Python / C++ / Java 实现。

## 3. 二分搜索

详解经典二分及其变种（左边界、右边界、旋转数组、峰值查找），分析 O(log n) 复杂度与循环不变量的正确性证明，讨论二分答案（参数搜索）的范式，附可视化与三语言实现。

## 4. 哈希查找

阐述哈希表 O(1) 平均查找的原理，简述哈希函数设计要点与冲突处理策略（详见 hashtable.md），讨论最坏 O(n) 退化的场景与一致性哈希，附三语言实现。

## 5. 广度优先搜索 (BFS)

讲解队列驱动的层次遍历过程，分析 O(V+E) 的时间复杂度，讨论 BFS 在无权最短路径、连通分量、层序遍历中的应用，附 Mermaid 状态图与三语言实现。

## 6. 深度优先搜索 (DFS)

阐述栈/递归驱动的深度探索过程，分析 O(V+E) 复杂度，讨论 DFS 在环检测、拓扑排序、连通分量、回溯剪枝中的应用，附递归树可视化与三语言实现。

## 7. 搜索策略对比

用表格对比线性/二分/哈希/BFS/DFS 的适用数据结构、时间复杂度、空间复杂度与典型应用场景，给出搜索策略选择的决策树。
