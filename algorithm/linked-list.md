## 1. 链表概述

对比链表与数组的内存模型差异（非连续 vs 连续），分析插入/删除/查找的时间复杂度对比，讨论缓存局部性对实际性能的影响。

## 2. 单链表

讲解单链表节点的定义与基本操作（头插、尾插、删除、查找），分析各操作 O(1) 或 O(n) 的复杂度，讨论哨兵节点（dummy head）简化边界处理的技巧，附 Python / C++ / Java 实现。

## 3. 双链表

阐述双链表的前驱指针带来的双向遍历能力，分析删除操作从 O(n) 降至 O(1) 的场景（已知节点指针时），讨论 LRU 缓存中双链表的应用，附三语言实现。

## 4. 环形链表

讲解尾节点指向头节点的循环链表结构，讨论约瑟夫环问题的数学解法 O(n) 与模拟解法 O(nm)，分析循环链表在操作系统进程调度中的建模，附三语言实现。

## 5. 经典操作与技巧

汇总快慢指针（找中点/判环）、反转链表（迭代/递归）、合并有序链表、链表分区等高频操作，分析每种操作的时间空间复杂度，附可视化与三语言实现。

## 6. 常见面试题型

分类整理 LeetCode 链表高频题：反转系列（LC-206/92/25）、合并系列（LC-21/23）、环检测（LC-141/142）、相交链表（LC-160）、回文链表（LC-234），给出解题模板与复杂度。
