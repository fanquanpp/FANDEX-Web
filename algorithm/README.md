## 1. 模块定位

本模块以**逻辑风**组织，每篇笔记聚焦三件事：**复杂度分析**、**可视化演示**、**多语言实现（Python / C++ / Java）**。适合作为面试复习速查手册或系统学习路线参考。

## 2. 约定

| 约定项 | 说明 |
|--------|------|
| 复杂度标注 | 统一使用 Θ / O / Ω / o / ω 五类渐进符号 |
| 代码语言 | 每个算法提供 Python 3、C++17、Java 17 三种实现 |
| 可视化 | 使用 ASCII 图或 Mermaid 流程图展示算法过程 |
| 题目链接 | LeetCode 题号以 `LC-xxxx` 格式标注 |
