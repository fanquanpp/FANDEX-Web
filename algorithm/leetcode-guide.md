## 1. 刷题方法论

阐述"理解 → 模仿 → 变式 → 创造"四阶段刷题法，强调每题必写复杂度分析、多解法对比与错题复盘，讨论"题海战术"与"精刷"的平衡。

## 2. 题型分类体系

按照数据结构（数组/链表/树/图/哈希）与算法范式（双指针/滑动窗口/二分/DP/贪心/回溯）双维度分类，给出分类树与各类别的题量建议。

## 3. 数组与字符串

汇总双指针、滑动窗口、前缀和、差分数组等高频技巧，精选 LC-1/3/11/15/42/56/239 等经典题目，给出解题模板与复杂度分析，附 Python / C++ / Java 实现。

## 4. 链表专题

汇总快慢指针、反转、合并、环检测等链表技巧，精选 LC-19/21/23/25/141/142/160/206 等题目，给出固定解题套路与边界条件检查清单，附三语言实现。

## 5. 树与图专题

汇总递归/迭代遍历、路径问题、LCA、拓扑排序等技巧，精选 LC-94/102/104/124/200/207/210 等题目，给出树问题递归思维训练方法，附三语言实现。

## 6. 动态规划专题

按"一维 DP → 二维 DP → 状态压缩 → 区间 DP"递进组织，精选 LC-70/72/121/152/198/300/322/72 等题目，给出状态定义与转移方程的推导模板，附三语言实现。

## 7. 回溯与搜索专题

汇总子集/排列/组合/N 皇后/数独等经典回溯问题，精选 LC-39/46/51/78/79/200 等题目，讲解剪枝策略与搜索空间可视化，附三语言实现。

## 8. 时间规划与节奏

给出 100 题 / 200 题 / 400 题三档刷题计划，按 Easy:Medium:Hard = 3:5:2 的比例分配，建议每日 2–3 题并配合周赛实战，附 Mermaid 甘特图。

## 9. 面试实战技巧

讲解白板编程的沟通策略（先说思路再写代码）、复杂度分析的口头表达模板、边界用例自检清单，以及"不会做"时的降级策略（暴力 → 优化 → 特殊情况）。
