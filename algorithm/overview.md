## 1. 算法与问题

定义什么是算法（有穷性、确定性、可行性），区分判定问题与优化问题，介绍问题规模 n 与输入编码的关系。

## 2. 渐进复杂度符号

系统讲解 Θ / O / Ω / o / ω 五类符号的严格定义与直观含义，配合函数增长速率可视化对比图，说明常见复杂度等级（O(1)、O(log n)、O(n)、O(n log n)、O(n²)、O(2ⁿ)）的实际意义。

## 3. 时空权衡

阐述"以空间换时间"与"以时间换空间"的经典策略，列举排序中辅助数组、搜索中索引表、DP 中备忘录等典型场景，给出权衡决策的量化框架。

## 4. 递归与主定理

讲解递归树方法与主定理（Master Theorem）三种情形，分析分治算法的递推式 T(n) = aT(n/b) + f(n)，通过可视化递归树展示每层工作量。

## 5. 摊还分析

介绍聚合分析、核算法与势能法三种摊还分析技术，以动态数组扩容、多重弹栈为例说明均摊 O(1) 的含义。

## 6. 算法设计范式总览

对比分治、贪心、动态规划、回溯、分支限界五大范式的适用条件与区别，用决策树图示展示范式选择流程。

## 7. 学习路线图

按"基础 → 进阶 → 专题"三阶段规划学习路径：基础阶段掌握排序与搜索，进阶阶段攻克 DP 与图论，专题阶段深入字符串、计算几何等方向。附 Mermaid 甘特图。
