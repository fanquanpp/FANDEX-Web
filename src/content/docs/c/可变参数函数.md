## 1. 可变参数

```c
#include <stdarg.h>

int sum(int count, ...) {
  va_list args;
  va_start(args, count);
  int total = 0;
  for (int i = 0; i < count; i++) {
    total += va_arg(args, int);
  }
  va_end(args);
  return total;
}

printf("%d\n", sum(3, 1, 2, 3)); // 6
```
