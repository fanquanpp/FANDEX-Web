## 1. setlocale

```c
#include <locale.h>

setlocale(LC_ALL, "zh_CN.UTF-8");
printf("宽字符: %lc\n", L'中');
```

## 2. 宽字符

```c
#include <wchar.h>

wchar_t *str = L"你好世界";
wprintf(L"%ls\n", str);
```
