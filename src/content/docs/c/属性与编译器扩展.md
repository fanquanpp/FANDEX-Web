## 1. **attribute**

```c
// 格式化检查
__attribute__((format(printf, 1, 2)))
void my_printf(const char *fmt, ...);

// 废弃标记
__attribute__((deprecated("use new_func instead")))
void old_func(void);

// 对齐
__attribute__((aligned(16))) int buffer[1024];

// 打包
struct __attribute__((packed)) Packed {
  char a;
  int b;
};
```
