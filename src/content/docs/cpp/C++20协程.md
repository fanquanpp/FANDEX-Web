## 1. 协程关键字

``co_await`、`co_yield`、`co_return`

```cpp
#include <coroutine>

Generator<int> fibonacci() {
  int a = 0, b = 1;
  while (true) {
    co_yield a;
    auto temp = a;
    a = b;
    b = temp + b;
  }
}

for (int n : fibonacci() | std::views::take(10)) {
  std::cout << n << ' ';
}
```
