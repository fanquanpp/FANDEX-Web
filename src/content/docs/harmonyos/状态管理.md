## 1. 状态装饰器

| 装饰器         | 说明         |
| -------------- | ------------ |
| `@State`       | 组件内状态   |
| `@Prop`        | 父子单向同步 |
| `@Link`        | 父子双向同步 |
| `@Provide`     | 跨层级提供   |
| `@Consume`     | 跨层级消费   |
| `@Watch`       | 监听变化     |
| `@StorageLink` | 持久化存储   |

```typescript
@Component
struct Child {
  @Link count: number

  build() {
    Button(`Count: ${this.count}`)
      .onClick(() => this.count++)
  }
}
```
