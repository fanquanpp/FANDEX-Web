## 1. 贝叶斯定理基础

### 1.1 贝叶斯公式

$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

在分类问题中：

$$P(y=c_k|\mathbf{x}) = \frac{P(\mathbf{x}|y=c_k) \cdot P(y=c_k)}{P(\mathbf{x})}$$

| 术语     | 符号            | 含义           |
| :------- | :-------------- | :------------- | -------------------- |
| 先验概率 | $P(y=c_k)$      | 类别的先验知识 |
| 似然     | $P(\mathbf{x}   | y=c_k)$        | 类别条件下特征的概率 |
| 后验概率 | $P(y=c_k        | \mathbf{x})$   | 观测数据后的类别概率 |
| 证据     | $P(\mathbf{x})$ | 归一化常数     |

### 1.2 贝叶斯决策规则

选择后验概率最大的类别：

$$\hat{y} = \arg\max_{c_k} P(y=c_k|\mathbf{x}) = \arg\max_{c_k} P(\mathbf{x}|y=c_k) \cdot P(y=c_k)$$

由于 $P(\mathbf{x})$ 对所有类别相同，可以省略。

## 2. 朴素贝叶斯分类器

### 2.1 条件独立性假设

**朴素**假设：给定类别时，各特征之间相互独立：

$$P(\mathbf{x}|y=c_k) = P(x_1, x_2, \ldots, x_d | y=c_k) = \prod_{j=1}^{d} P(x_j | y=c_k)$$

### 2.2 分类公式

$$\hat{y} = \arg\max_{c_k} P(y=c_k) \prod_{j=1}^{d} P(x_j | y=c_k)$$

**对数形式**（防止下溢）：

$$\hat{y} = \arg\max_{c_k} \left[\ln P(y=c_k) + \sum_{j=1}^{d} \ln P(x_j | y=c_k)\right]$$

### 2.3 参数估计

**先验概率**：

$$P(y=c_k) = \frac{|D_{c_k}|}{|D|} = \frac{\text{类别}c_k\text{的样本数}}{\text{总样本数}}$$

**条件概率**（离散特征）：

$$P(x_j = a_{jl} | y=c_k) = \frac{|D_{c_k, x_j=a_{jl}}|}{|D_{c_k}|}$$

## 3. 拉普拉斯平滑

### 3.1 零概率问题

如果某个特征值在训练集中未出现，则 $P(x_j|y=c_k) = 0$，导致整个乘积为零：

$$P(\mathbf{x}|y=c_k) = \prod_{j=1}^d P(x_j|y=c_k) = 0 \quad \text{（任一项为零则全为零）}$$

### 3.2 拉普拉斯平滑

在分子加 $\lambda$，分母加 $\lambda S_j$（$S_j$ 为特征 $j$ 的取值数）：

$$P(x_j = a_{jl} | y=c_k) = \frac{|D_{c_k, x_j=a_{jl}}| + \lambda}{|D_{c_k}| + \lambda S_j}$$

- $\lambda = 1$：标准拉普拉斯平滑
- $\lambda = 0$：无平滑（最大似然估计）
- $\lambda > 0$：平滑强度可调

**先验概率平滑**：

$$P(y=c_k) = \frac{|D_{c_k}| + \lambda}{|D| + \lambda K}$$

## 4. 朴素贝叶斯变体

### 4.1 高斯朴素贝叶斯

适用于**连续特征**，假设特征服从高斯分布：

$$P(x_j | y=c_k) = \frac{1}{\sqrt{2\pi\sigma_{kj}^2}} \exp\left(-\frac{(x_j - \mu_{kj})^2}{2\sigma_{kj}^2}\right)$$

参数估计：

$$\mu_{kj} = \frac{1}{|D_{c_k}|}\sum_{\mathbf{x}_i \in D_{c_k}} x_{ij}$$

$$\sigma_{kj}^2 = \frac{1}{|D_{c_k}|}\sum_{\mathbf{x}_i \in D_{c_k}} (x_{ij} - \mu_{kj})^2$$

### 4.2 多项式朴素贝叶斯

适用于**离散计数特征**（如文本分类中的词频）：

$$P(x_j | y=c_k) = \frac{N_{kj} + \alpha}{N_k + \alpha |V|}$$

其中 $N_{kj}$ 为类别 $c_k$ 中特征 $j$ 出现的总次数，$|V|$ 为词汇表大小。

### 4.3 伯努利朴素贝叶斯

适用于**二值特征**（特征出现/不出现）：

$$P(\mathbf{x} | y=c_k) = \prod_{j=1}^{d} P(x_j | y=c_k)^{b_j} [1 - P(x_j | y=c_k)]^{1-b_j}$$

其中 $b_j \in \{0, 1\}$ 表示特征 $j$ 是否出现。

### 4.4 变体对比

| 变体     | 特征类型 | 分布假设   | 典型应用        |
| :------- | :------- | :--------- | :-------------- |
| 高斯NB   | 连续     | 正态分布   | 数值型分类      |
| 多项式NB | 计数     | 多项分布   | 文本分类        |
| 伯努利NB | 二值     | 伯努利分布 | 短文本/二值特征 |

## 5. 朴素贝叶斯优缺点

### 5.1 优点

- **训练速度快**：$O(n \cdot d)$，只需计算频率
- **预测速度快**：$O(d \cdot K)$
- **小样本友好**：参数少，不易过拟合
- **高维数据**：特征维度高时仍有效
- **增量学习**：可在线更新模型

### 5.2 缺点

- **条件独立性假设过强**：特征间往往存在依赖
- **概率估计不准**：输出概率不可靠（但分类结果通常可用）
- **对输入数据格式敏感**：需选择合适的变体

### 5.3 改进方向

| 方法         | 说明                             |
| :----------- | :------------------------------- |
| 半朴素贝叶斯 | 放松独立性假设，如AODE、TAN      |
| 贝叶斯网络   | 用图结构表达特征依赖关系         |
| 特征选择     | 移除冗余特征，使独立性假设更合理 |
