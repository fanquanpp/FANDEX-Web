## 1. 最大间隔分类器

SVM 的核心思想是寻找一个超平面，使得**间隔（Margin）最大化**。

### 1.1 间隔定义

给定训练集 $\{(\mathbf{x}_i, y_i)\}_{i=1}^n$，$y_i \in \{-1, +1\}$，超平面为：

$$\mathbf{w}^T\mathbf{x} + b = 0$$

**函数间隔**：

$$\hat{\gamma}_i = y_i(\mathbf{w}^T\mathbf{x}_i + b)$$

**几何间隔**：

$$\gamma_i = \frac{y_i(\mathbf{w}^T\mathbf{x}_i + b)}{\|\mathbf{w}\|}$$

### 1.2 优化问题

最大化几何间隔等价于：

$$\max_{\mathbf{w}, b} \frac{2}{\|\mathbf{w}\|} \quad \text{s.t.} \quad y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1, \quad i = 1, \ldots, n$$

等价于：

$$\min_{\mathbf{w}, b} \frac{1}{2}\|\mathbf{w}\|^2 \quad \text{s.t.} \quad y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1$$

### 1.3 对偶问题

引入拉格朗日乘子 $\alpha_i \geq 0$，构建拉格朗日函数：

$$\mathcal{L}(\mathbf{w}, b, \boldsymbol{\alpha}) = \frac{1}{2}\|\mathbf{w}\|^2 - \sum_{i=1}^n \alpha_i [y_i(\mathbf{w}^T\mathbf{x}_i + b) - 1]$$

对偶问题：

$$\max_{\boldsymbol{\alpha}} \sum_{i=1}^n \alpha_i - \frac{1}{2}\sum_{i=1}^n\sum_{j=1}^n \alpha_i \alpha_j y_i y_j \mathbf{x}_i^T\mathbf{x}_j$$

$$\text{s.t.} \quad \alpha_i \geq 0, \quad \sum_{i=1}^n \alpha_i y_i = 0$$

**KKT条件**：

$$\alpha_i [y_i(\mathbf{w}^T\mathbf{x}_i + b) - 1] = 0$$

- $\alpha_i > 0$ 的样本点称为**支持向量**
- 支持向量位于间隔边界上：$y_i(\mathbf{w}^T\mathbf{x}_i + b) = 1$
- 通常支持向量只占训练样本的**少数**

## 2. 核函数

### 2.1 核技巧

当数据线性不可分时，将数据映射到高维空间：

$$\phi: \mathbb{R}^d \rightarrow \mathbb{R}^D \quad (D \gg d)$$

对偶问题中只出现**内积** $\phi(\mathbf{x}_i)^T\phi(\mathbf{x}_j)$，可用核函数替代：

$$K(\mathbf{x}_i, \mathbf{x}_j) = \phi(\mathbf{x}_i)^T\phi(\mathbf{x}_j)$$

**优势**：无需显式计算高维映射，只需计算核函数值。

### 2.2 常用核函数

| 核函数    | 公式                                                                   | 参数           | 适用场景     |
| :-------- | :--------------------------------------------------------------------- | :------------- | :----------- |
| 线性核    | $K(\mathbf{x}, \mathbf{z}) = \mathbf{x}^T\mathbf{z}$                   | 无             | 线性可分     |
| 多项式核  | $K(\mathbf{x}, \mathbf{z}) = (\gamma \mathbf{x}^T\mathbf{z} + r)^d$    | $\gamma, r, d$ | 非线性       |
| RBF核     | $K(\mathbf{x}, \mathbf{z}) = \exp(-\gamma\|\mathbf{x}-\mathbf{z}\|^2)$ | $\gamma$       | 通用         |
| Sigmoid核 | $K(\mathbf{x}, \mathbf{z}) = \tanh(\gamma \mathbf{x}^T\mathbf{z} + r)$ | $\gamma, r$    | 神经网络模拟 |

**RBF核**是最常用的核函数：

$$K(\mathbf{x}, \mathbf{z}) = \exp\left(-\frac{\|\mathbf{x}-\mathbf{z}\|^2}{2\sigma^2}\right)$$

- $\gamma = \frac{1}{2\sigma^2}$ 越大，核越"尖锐"，模型越复杂
- $\gamma$ 越小，核越"平滑"，模型越简单

### 2.3 Mercer条件

核函数 $K$ 必须满足**Mercer条件**：对任意函数 $f$，

$$\int\int K(\mathbf{x}, \mathbf{z})f(\mathbf{x})f(\mathbf{z})d\mathbf{x}d\mathbf{z} \geq 0$$

等价于核矩阵**半正定**。

## 3. 软间隔

### 3.1 松弛变量

当数据存在噪声或不可完全线性分离时，引入松弛变量 $\xi_i \geq 0$：

$$\min_{\mathbf{w}, b, \boldsymbol{\xi}} \frac{1}{2}\|\mathbf{w}\|^2 + C\sum_{i=1}^n \xi_i$$

$$\text{s.t.} \quad y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1 - \xi_i, \quad \xi_i \geq 0$$

- $C$ 为惩罚参数，控制间隔与误分类的权衡
- $C \to \infty$：硬间隔，不允许误分类
- $C \to 0$：允许更多误分类，间隔更大

### 3.2 对偶问题

$$\max_{\boldsymbol{\alpha}} \sum_{i=1}^n \alpha_i - \frac{1}{2}\sum_{i,j} \alpha_i \alpha_j y_i y_j K(\mathbf{x}_i, \mathbf{x}_j)$$

$$\text{s.t.} \quad 0 \leq \alpha_i \leq C, \quad \sum \alpha_i y_i = 0$$

### 3.3 支持向量分类

| 条件                      | 样本位置       | $\alpha_i$         |
| :------------------------ | :------------- | :----------------- |
| $y_i f(\mathbf{x}_i) > 1$ | 间隔外         | $\alpha_i = 0$     |
| $y_i f(\mathbf{x}_i) = 1$ | 间隔边界上     | $0 < \alpha_i < C$ |
| $y_i f(\mathbf{x}_i) < 1$ | 间隔内或误分类 | $\alpha_i = C$     |

## 4. SMO算法

SMO（Sequential Minimal Optimization）是求解SVM对偶问题的高效算法。

### 4.1 基本思想

每次只优化**两个**拉格朗日乘子 $\alpha_i, \alpha_j$，解析求解子问题：

1. 选择一对 $\alpha_i, \alpha_j$
2. 固定其他 $\alpha_k$，求解 $\alpha_i, \alpha_j$ 的最优值
3. 更新偏置 $b$
4. 重复直到收敛

### 4.2 两变量优化

由约束 $\sum \alpha_i y_i = 0$，可得：

$$\alpha_i y_i + \alpha_j y_j = -\sum_{k \neq i,j} \alpha_k y_k = \zeta$$

$\alpha_j$ 的更新公式：

$$\alpha_j^{new} = \alpha_j^{old} + \frac{y_j(E_i - E_j)}{\eta}$$

其中 $E_i = f(\mathbf{x}_i) - y_i$ 为预测误差，$\eta = K(\mathbf{x}_i, \mathbf{x}_i) + K(\mathbf{x}_j, \mathbf{x}_j) - 2K(\mathbf{x}_i, \mathbf{x}_j)$。

**裁剪**：

$$\alpha_j^{new,clipped} = \text{clip}(\alpha_j^{new}, L, H)$$

$$\alpha_i^{new} = \alpha_i^{old} + y_i y_j (\alpha_j^{old} - \alpha_j^{new,clipped})$$

### 4.3 启发式选择

- **外循环**：遍历所有样本，选择违反KKT条件的 $\alpha_i$
- **内循环**：选择使 $|E_i - E_j|$ 最大的 $\alpha_j$
