## 1. 线性回归模型

线性回归是最基本的回归模型，假设目标变量与特征之间存在**线性关系**。

### 1.1 模型定义

$$\hat{y} = \mathbf{w}^T \mathbf{x} + b = w_1 x_1 + w_2 x_2 + \cdots + w_d x_d + b$$

其中 $\mathbf{w} \in \mathbb{R}^d$ 为权重向量，$b$ 为偏置项。

**向量化形式**（$n$ 个样本）：

$$\hat{\mathbf{y}} = \mathbf{X}\mathbf{w} + b\mathbf{1}$$

其中 $\mathbf{X} \in \mathbb{R}^{n \times d}$，$\mathbf{w} \in \mathbb{R}^d$。

### 1.2 几何解释

线性回归在特征空间中寻找一个**超平面**，使得所有样本点到超平面的距离之和最小：

$$d = \frac{|y_i - \hat{y}_i|}{\sqrt{1 + \|\mathbf{w}\|^2}}$$

## 2. 最小二乘法

### 2.1 代价函数

均方误差（MSE）代价函数：

$$J(\mathbf{w}, b) = \frac{1}{2n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2 = \frac{1}{2n} \|\mathbf{y} - \mathbf{X}\mathbf{w}\|^2$$

### 2.2 解析解

对 $J$ 关于 $\mathbf{w}$ 求导并令其为零：

$$\frac{\partial J}{\partial \mathbf{w}} = -\frac{1}{n}\mathbf{X}^T(\mathbf{y} - \mathbf{X}\mathbf{w}) = 0$$

得到**正规方程**：

$$\mathbf{X}^T\mathbf{X}\mathbf{w} = \mathbf{X}^T\mathbf{y}$$

当 $\mathbf{X}^T\mathbf{X}$ 可逆时，解析解为：

$$\mathbf{w}^* = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{y}$$

**解析解的局限性**：

- $\mathbf{X}^T\mathbf{X}$ 不可逆（特征共线性）
- 计算复杂度 $O(d^3 + d^2 n)$，高维时不可行
- 大规模数据无法全部装入内存

## 3. 梯度下降

### 3.1 梯度下降原理

沿负梯度方向迭代更新参数：

$$\mathbf{w}_{t+1} = \mathbf{w}_t - \alpha \nabla_{\mathbf{w}} J(\mathbf{w}_t)$$

其中 $\alpha$ 为学习率，梯度为：

$$\nabla_{\mathbf{w}} J = \frac{1}{n} \mathbf{X}^T(\mathbf{X}\mathbf{w} - \mathbf{y})$$

### 3.2 梯度下降变体

| 变体                   | 每次更新使用样本       | 优点                   | 缺点           |
| :--------------------- | :--------------------- | :--------------------- | :------------- |
| 批量梯度下降（BGD）    | 全部 $n$ 个            | 稳定收敛               | 计算慢         |
| 随机梯度下降（SGD）    | 1个                    | 速度快、可跳出局部最优 | 波动大         |
| 小批量梯度下降（MBGD） | $m$个（$1 < m \ll n$） | 兼顾速度和稳定性       | 需调batch size |

**SGD更新规则**：

$$\mathbf{w}_{t+1} = \mathbf{w}_t - \alpha (h_{\mathbf{w}}(\mathbf{x}_i) - y_i)\mathbf{x}_i$$

### 3.3 学习率策略

| 策略       | 公式                                                      | 说明             |
| :--------- | :-------------------------------------------------------- | :--------------- |
| 固定学习率 | $\alpha_t = \alpha_0$                                     | 简单但可能不收敛 |
| 阶梯衰减   | $\alpha_t = \alpha_0 \cdot \gamma^{\lfloor t/T \rfloor}$  | 每T步衰减        |
| 指数衰减   | $\alpha_t = \alpha_0 \cdot e^{-kt}$                       | 平滑衰减         |
| 余弦退火   | $\alpha_t = \frac{\alpha_0}{2}(1 + \cos(\frac{t\pi}{T}))$ | 周期性调整       |
| Warmup     | 前N步线性增加                                             | 避免初期不稳定   |

### 3.4 高级优化器

| 优化器   | 核心思想           | 公式                                                      |
| :------- | :----------------- | :-------------------------------------------------------- |
| Momentum | 累积历史梯度       | $\mathbf{v}_t = \beta \mathbf{v}_{t-1} + \alpha \nabla J$ |
| AdaGrad  | 自适应学习率       | $\alpha_t = \frac{\alpha_0}{\sqrt{G_t + \epsilon}}$       |
| RMSProp  | 指数移动平均       | $E[g^2]_t = \beta E[g^2]_{t-1} + (1-\beta)g_t^2$          |
| Adam     | Momentum + RMSProp | 结合一阶和二阶矩估计                                      |

## 4. 正则化

### 4.1 Ridge回归（L2正则化）

$$J_{Ridge}(\mathbf{w}) = \frac{1}{2n}\|\mathbf{y} - \mathbf{X}\mathbf{w}\|^2 + \lambda \|\mathbf{w}\|^2$$

解析解：

$$\mathbf{w}^*_{Ridge} = (\mathbf{X}^T\mathbf{X} + \lambda \mathbf{I})^{-1}\mathbf{X}^T\mathbf{y}$$

- $\lambda > 0$ 时，$\mathbf{X}^T\mathbf{X} + \lambda \mathbf{I}$ 一定可逆
- 参数被**收缩**到零附近，但不会恰好为零

### 4.2 Lasso回归（L1正则化）

$$J_{Lasso}(\mathbf{w}) = \frac{1}{2n}\|\mathbf{y} - \mathbf{X}\mathbf{w}\|^2 + \lambda \|\mathbf{w}\|_1$$

- 产生**稀疏解**，部分参数恰好为零
- 等价于带 $L_1$ 约束的优化问题
- 无解析解，需用坐标下降法求解

### 4.3 正则化几何解释

```
L2正则化: 约束区域为圆形 → 与等高线相切于轴附近
L1正则化: 约束区域为菱形 → 与等高线相切于轴上（稀疏解）
```

$$\text{L2约束}: \|\mathbf{w}\|^2 \leq t \quad \text{(圆形)}$$
$$\text{L1约束}: \|\mathbf{w}\|_1 \leq t \quad \text{(菱形)}$$

## 5. 模型评估

### 5.1 回归评估指标

| 指标           | 公式                                                        | 说明         |
| :------------- | :---------------------------------------------------------- | :----------- |
| MAE            | $\frac{1}{n}\sum\|y_i - \hat{y}_i\|$                        | 平均绝对误差 |
| MSE            | $\frac{1}{n}\sum(y_i - \hat{y}_i)^2$                        | 均方误差     |
| RMSE           | $\sqrt{\text{MSE}}$                                         | 均方根误差   |
| $R^2$          | $1 - \frac{\sum(y_i - \hat{y}_i)^2}{\sum(y_i - \bar{y})^2}$ | 决定系数     |
| Adjusted $R^2$ | $1 - \frac{(1-R^2)(n-1)}{n-d-1}$                            | 修正决定系数 |

### 5.2 假设检验

**t检验**：检验单个回归系数是否显著

$$t_j = \frac{\hat{w}_j}{\text{SE}(\hat{w}_j)} \sim t_{n-d-1}$$

**F检验**：检验模型整体是否显著

$$F = \frac{\text{ESS}/d}{\text{RSS}/(n-d-1)} \sim F_{d, n-d-1}$$

### 5.3 线性回归假设

| 假设         | 含义                   | 检验方法            |
| :----------- | :--------------------- | :------------------ |
| 线性性       | 因变量与自变量线性相关 | 残差图              |
| 独立性       | 残差相互独立           | Durbin-Watson检验   |
| 同方差性     | 残差方差恒定           | Breusch-Pagan检验   |
| 正态性       | 残差服从正态分布       | Q-Q图、Shapiro检验  |
| 无多重共线性 | 自变量间不高度相关     | VIF（方差膨胀因子） |
