## 1. 泛型函数

```go
func Map[T any, R any](s []T, f func(T) R) []R {
  result := make([]R, len(s))
  for i, v := range s { result[i] = f(v) }
  return result
}
```

## 2. 类型约束

```go
type Number interface {
  ~int | ~int8 | ~int16 | ~int32 | ~int64 |
  ~float32 | ~float64
}

func Sum[T Number](nums []T) T {
  var total T
  for _, n := range nums { total += n }
  return total
}
```
