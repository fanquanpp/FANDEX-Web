## 1. C# 12 新特性

- 主构造函数：`class MyClass(int x) { }`
- 集合表达式：`int[] arr = [1, 2, 3];`
- 内联数组
- ref readonly 参数

## 2. C# 13 新特性

- `params` 集合
- 部分属性
- 扩展类型
- 锁对象
