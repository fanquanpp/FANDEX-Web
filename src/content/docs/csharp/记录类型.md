## 1. record

```csharp
public record Person(string Name, int Age);

var p1 = new Person("Alice", 25);
var p2 = p1 with { Age = 26 }; // 不可变修改
Console.WriteLine(p1 == p2);    // false（值相等）
```

## 2. record struct
