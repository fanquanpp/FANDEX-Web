## 1. 项目简介

- Java → MySQL

## 4. 设计原则

1. **渐进式**: 每个模块从零开始，循序渐进
2. **可验证**: 每个知识点配有可运行的代码示例
3. **可关联**: 跨模块引用建立知识网络
4. **可练习**: 每个模块有配套练习题
5. **个性化**: 各技术领域用最适合的风格呈现
6. **最新性**: 补充 2026 年最新内容
